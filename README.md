# Collection of Web Safe CSS Fonts

## Sans serif fonts
**Font Family: Arial**

`body {
     font-family: Arial, "Helvetica Neue", Helvetica, sans-serif; 
     font-size: 16px;
     color: #000;
}`<br/><br/>

**Font Family: Arial Black**

`body {
     font-family: "Arial Black", "Arial Bold", Gadget, sans-serif;
     font-size: 16px;
     color: #000;
}`<br/><br/>

**Font Family: Arial Narrow**

`body {
     font-family: "Arial Black", "Arial Bold", Gadget, sans-serif;
     font-size: 16px;
     color: #000;
}`<br/><br/>
